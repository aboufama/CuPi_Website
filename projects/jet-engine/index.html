Jet Engine

High performance in-house designed jet propulsion system prototype for efficient and compact performance.

Drag to explore the 3D model

Our Jet Engine project represents a significant engineering challenge in developing a high-performance propulsion system optimized for small-scale UAV platforms. The system is entirely custom-designed, from the compressor stages to the combustion chamber, enabling unprecedented control over performance characteristics.

The fully custom fuel injection system has been engineered for future integration with novel oxidizer injection technologies, opening pathways to advanced propulsion concepts. This project pushes the boundaries of what's possible in compact jet engine design, balancing efficiency, power output, and form factor constraints.

The prototype serves as both a research platform and a proof-of-concept for next-generation propulsion systems.
